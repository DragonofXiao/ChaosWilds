# Chaos Wilds

混沌世界 - 一款 2D 俯视角类 Rogue 游戏的 Demo 版本

## 游戏简介

- 核心玩法：每层随机部位组合 + 肉鸽升级
- 战斗形式：2D 俯视角，类以撒
- 攻击模式：半自动（自动普攻 + 手动部位技能）
- 当前版本：Demo V1


## 技术栈

- 引擎：Godot 4.x
- 开发语言：GDScript
- 数据格式：CSV


## 运行方法
